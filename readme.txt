== Description ==

The WordPress Ad Manager plugin is a robust advert manager for WordPress sites that allows you to control and distribute advertisements from a central dashboard.

Download it now, or head over to the docs for a step-by-step install and how-to.

## Features Include

*   Public & private ads
*   Image adverts
*   WordPress 4.7.2 compatible
*   Date & time scheduling
*   Administration control
*   Simple dashboard UI
*   Supports Google Analytics UTM tracking code
*   Widgets and hooks that allow you to display your adverts anywhere
*   Itâ€™s also developer friendly, with tonnes of hooks and filters

== Installation ==

1. Download the most up to date copy of the plugin
1. Head over to your WordPress Dashboard and select Plugins > Add New
1. Select Upload Plugin, then upload your downloaded zip file
1. Select Activate Plugin


== Frequently Asked Questions ==

= Who can upload adverts? =

You can set who can upload adverts in the plugin's settings.

= Can I schedule ads? =

Ofcourse! You can schedule ads to a set period of time, after mid-day, whatever you want!

= Can I add adverts to the sidebar on my site? =

Yehp, you can adverts to anypart of your site through the use of hooks, shortcodes and widgets. 

= Are there developer hooks? =

Yehp! Head over to our documentation guides for available actions and filters

= Can I upload HTML adverts? =

Sadly, not yet. But we're working on it...
